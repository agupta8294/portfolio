Image Manipulation

Computer Science Principles

February 25, 2019 - March 28, 2019

The objective of this project was to code a program that manipulates images. In the class, we were given a choice to make the program for 4 different clients. We chose Client 4 - A Developer. For this, we added a border, added border size input, radius input, thick mode (which makes the lines darker and wider), and transparency. We made all of these parameters in a function. We also made it so that all images had rounded corners. Our documentation for this project is right here. We used this project to develop our skills of using Python, debugging code, and changing images. One challenge I overcame was when the border made an '8' shape around the smaller images. I overcame this by adding a barrier which the border could not pass based on the size of the image.
