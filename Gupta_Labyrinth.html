The Labyrinth

Computer Science Principles

August 30, 2018 - September 9, 2018

The objective of this project was to create a game using Scratch. My partner (Kaden Fox) and I decided to create a Labyrinth game in which players would go through a maze and try to find their way out. I programmed this project while my partner mainly focused on documenting the project. I used this project to develop my skills of leadership and creative thinking. One big challenge I overcame was when we could not find a way to get the maze model (which was created with Microsoft Paint) to become a sprite in Scratch. I figured out a way to do this by taking a screenshot of the maze and uploading the photo to Scratch.
